[Dojo](https://dojo.io/home) is a progressive framework for modern web apps. It provides a holistic approach to engineering modern web applications while remaining progressively modular in which aspects a project may utilize.

## Step 1: Set Up Your Dojo Project

Get started by globally installing the `@dojo/cli` and `@dojo/cli-create-app` packages.

<Snippet width="100%"dark text="npm i -g @dojo/cli @dojo/cli-create-app" />
<Caption>Globally installing the <InlineCode>@dojo/cli</InlineCode> and <InlineCode>@dojo/cli-create-app</InlineCode> packages with <Link href="https://www.npmjs.com">npm</Link>.</Caption>

Next, set up a Dojo app with the `dojo create` command and move into the project directory:

<Snippet width="100%"dark text="dojo create app --name my-dojo-app && cd my-dojo-app" />
<Caption>Creating a Dojo app with the Dojo CLI.</Caption>
