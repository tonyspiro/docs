## Init

The `vercel init` command is used to initialize projects locally from the examples found in the [<ProductName /> repository](https://github.com/vercel/vercel/tree/master/examples).

### Basic Usage

<Snippet width="100%"dark text="vercel init" />
<Caption>Using the <InlineCode>vercel init</InlineCode> command to initialize a project locally.</Caption>

### Extended Usage

<Snippet width="100%"dark text="vercel init [project-name]" />
<Caption>Using the <InlineCode>vercel init</InlineCode> command to initialize a specific project locally.</Caption>

<Snippet width="100%"dark text="vercel init [project-name] [new-project-name]" />
<Caption>Using the <InlineCode>vercel init</InlineCode> command to initialize a specific project locally and rename the directory.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `vercel init` command:

- `--debug`
- `--help`
- `--force`

For more information on global options and their usage, refer to the [options section](#options).
