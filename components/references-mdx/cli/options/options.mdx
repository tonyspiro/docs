# Options

This section contains information about the two types of options available to pass when using [<ProductShortName /> CLI](/download) commands, [unique](#unique-options) and [global](#global-options).

To understand which options can be used with a particular command, take a look at the specific command for more information.

## Unique Options

Unique options relate only to a single <ProductShortName /> CLI command and as such, are documented in full in the command section they relate to.

When using unique options, some of the shorthands may conflict with those of [global options](#global-options), this only occurs when the particular global option is not available for that command.

## Global Options

Global options are commonly available to use with multiple <ProductShortName /> CLI commands.

### Debug

The `--debug` option, shorthand `-d`, can be used to provide a more verbose output when running <ProductShortName /> CLI commands.
